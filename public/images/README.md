# Структура медиа файлов

Разместите ваши изображения и GIF анимации в следующей структуре:

## Изображения историй

```
/public/images/
  story1/
    background.jpg    - Фоновое изображение для истории 1
    character.gif     - GIF анимация персонажа
    magic-item.gif    - GIF анимация магического предмета
    ...
  story2/
    background.jpg
    door.gif
    ...
  story3/
    ...
  ...
  story8/
    background.jpg
    ...
```

## Аудио файлы

```
/public/audio/
  story1.mp3          - Фоновая музыка для истории 1
  story2.mp3
  ...
  story8.mp3
  effects/
    click.mp3         - Звуковые эффекты
    door-open.mp3
    ...
```

## Рекомендации

- **Фоновые изображения**: 1920x1080 или выше
- **GIF анимации**: оптимизированные для веба (< 5MB каждый)
- **Аудио треки**: MP3, 128-320 kbps
- **Звуковые эффекты**: короткие MP3 файлы (< 1MB)
